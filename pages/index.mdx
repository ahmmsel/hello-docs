# Introduction

Hello Platform is made to provide a platform for users to share their **opinions** and connect with others who share their interests.

## Features

Pepole can post their opinions and reply or follow each other, they also can edit their profile such as add profile picture and cover image, edit bio, etc..
also the users will able to delete their profile from platform, the platform contains much more features you can vist it

[Preview Hello Platform](https://hello-eight-rho.vercel.app/)

## Tech Stack

- Typescript
- React
- NextJs
- Next Auth
- NodeJs
- MongoDB
- Prisma ORM
- Tailwind CSS
- Zustand For State Management
- FontAwesome

## Quick Start

First make sure you already have **Node js** installed on your machine.
After unzipping the main file, you will find a folder called **resource**, open this
folder through **VS Code** or any other code editor you prefer, and then type the
following command inside the terminal.
